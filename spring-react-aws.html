Spring React AWS Project

This is a Spring React Amazon Web Services full stack application.

This is a java project that I've made for practice purposes and just for fun. It is a webapplication where you can upload a profile image. The profile image is uploaded in the front end of the application and then stored in the cloud, wich is an Amazon Web Services S3 bucket. I've used Spring Boot for the backend of the application and React for the frontend. It is configered with a in-memory database and AWS S3 cloud Storage. Other technologies that I have used are: Apach Maven, HTML/CSS, React-dropzone
